# 📚 简介

该插件用于美化 [你的日记网页版](https://nideriji.cn/login/) 以获得更好的体验

需要浏览器装有 [Tampermonkey](https://tampermonkey.net/) 插件，创建插件并将 [这个文件](Nideriji-Beautify.js) 内的代码复制即可
