# 📚 简介

该插件能使浏览器自动跳跃到从电脑版 QQ 聊天消息内点击的网址，跳过脑残的电脑管家拦截

# 📦 使用方式

需要浏览器装有 [Tampermonkey](https://tampermonkey.net/) 插件，下方表格中挑一个链接安装

| 正式版                                                 |
| ------------------------------------------------------ |
| [安装](https://greasyfork.org/zh-CN/scripts/448283-qq) |
